# 📋 ST-Chat-Exporter

Удобное и функциональное расширение для SillyTavern, которое позволяет гибко экспортировать, фильтровать и сохранять логи ваших чатов. Отлично подходит для архивации длинных ролевых сюжетов и удобного чтения логов вне браузера.

## ✨ Особенности

* **Гибкий выбор сообщений:** Экспортируйте весь чат, заданный диапазон (например, с 10 по 50 сообщение) или выбирайте конкретные реплики через запятую.

  <img width="808" height="163" alt="изображение" src="https://github.com/user-attachments/assets/81febf7d-694c-4de1-9e91-1c4733ba6d7f" />

* **Фильтрация по автору:** Можно выгрузить только реплики пользователя или только ответы бота.
* **Умный поиск:** Встроенный поиск по тексту с подсветкой найденных слов прямо в окне предпросмотра.

  <img width="789" height="187" alt="изображение" src="https://github.com/user-attachments/assets/58ce1dea-821d-4e08-9466-56878f5e4267" />

* **Разные форматы экспорта:**
  * **TXT:** Чистый текст (с возможностью включить отступы, имена, номера сообщений и таймстампы).
  * **HTML:** Красиво оформленная веб-страница со стилизованным текстом (шрифты Crimson Pro / JetBrains Mono) для комфортного чтения.

<img width="1036" height="939" alt="изображение" src="https://github.com/user-attachments/assets/bd74f854-d083-4c24-b0db-b490855c91ac" />

* **Копирование в буфер:** Мгновенное копирование выбранного фрагмента лога в один клик.
* **Адаптивный интерфейс:** Окно экспортера можно свободно перемещать по экрану, оно поддерживает изменение размера и корректно работает как на ПК, так и на мобильных устройствах.

## 🛠️ Установка / Обновление

1. Зайдите в **Extensions** -> **Install Extension**.
2. Используйте ссылку: `https://github.com/Gromova-Olga/ST-Chat-Exporter`
3. Если расширение уже установлено — нажмите **Update** во вкладке расширений.
